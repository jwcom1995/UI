문단을 나누는 태그들

p : 문단 영역을 나누는 태그로는 p태그와 pre태그가 있다. p태그는 문단영역얼 나누는 태그이지만 한 개의 공백만 표시하며 줄 바꿈 입력을 별도의 태그로 지정해 주어야 한다. pre태그는 여러칸 띄우기 혹은 줄 바꿈 등을 포함하여 입력한 내용 그대로 표현하는 태그이다.

그 밖에 텍스트를 다루는 태그들

b : 글자를 굵게 표시하는 태그
strong : 글자를 굵게 표시하는 태그
i : 글자를 기울이는 태그
em : 글자를 기울이는 태그
q : 인용 문구를 나타내는 태그
blackquote : 인용문구를 나타내는 태그
mark : 형광펜 효과를 나타내는 태그
u : 글자에 밑줄을 긋는 태그
기본 글자에 sub : 아래 첨자를 나타내는 태그와 sup : 윗 첨자를 나타내는 태그이다
small : 글자를 작게표시하는 태그
s : 글자에 취소선을 넣는 태그
